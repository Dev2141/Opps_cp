Hide Message

Embed any text or file inside a PNG image using LSB steganography. Your secret is encrypted with AES-256-GCM before embedding — completely invisible to the naked eye.

  • Supports text and file payloads
  • AES-256-GCM password encryption
  • Live capacity indicator
  • Output always lossless PNG

Extract Message

Recover hidden content from a stego PNG or BMP image. Enter the decryption password to unlock and reveal the embedded text message or reconstruct the original file.

  • Decrypts AES-256-GCM payload
  • Auto-detects text vs file type
  • One-click copy or download
  • JPEG lossy-compression warning

Secure Share

Send and receive stego images through named mailboxes on a local Java API server. Collaborate securely with your team — messages are protected end-to-end with a shared passphrase.

  • Named mailbox system
  • Passphrase-protected inbox
  • Send, receive, delete messages
  • Auto-loads into Extract tab

Stego Analyzer

Run statistical analysis on any image to detect steganographic modifications. Measures LSB distribution, chi-square anomalies, entropy, and bit transition patterns with a confidence score.

  • LSB ratio per RGB channel
  • Chi-square statistical test
  • Entropy & transition analysis
  • Confidence verdict score
Open Analyzer
HOW IT WORKS
01

Prepare Your Secret

Write your secret text or select any file you want to hide. Choose a strong password — this drives PBKDF2 key derivation for AES-256-GCM encryption before any pixel is touched.

02

Embed in an Image

Upload any PNG or BMP as a carrier. The encrypted payload is written into the Least Significant Bits of each pixel's RGB channels — changes imperceptible to humans, but retrievable by algorithm.

03

Download & Share

Download the stego PNG — it looks identical to the original. Share it via the Secure Share mailbox or any channel. The recipient runs Extract with the same password to recover the secret.
